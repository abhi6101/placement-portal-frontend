Introduction to Data Science

Unlock the power of data by mastering analysis, visualization, and machine learning with Python.

Course Overview

This course is your entry point into the exciting field of data science. You will learn the entire data science pipeline, from data collection and cleaning to analysis, visualization, and building your first predictive models using Python's powerful libraries.

What You'll Master

  • Data manipulation and analysis using Pandas.
  • Numerical computing and array operations with NumPy.
  • Creating insightful data visualizations with Matplotlib and Seaborn.
  • Understanding the fundamentals of machine learning.
  • Building and evaluating regression and classification models with Scikit-learn.
  • Communicating results and findings effectively.

8-Week Curriculum

Weeks 1-2: Python & Data Wrangling

Python basics for data science, plus cleaning and preparing data with NumPy and Pandas.

Weeks 3-4: Data Analysis & Visualization

Exploratory data analysis (EDA) and creating compelling charts with Matplotlib/Seaborn.

Weeks 5-6: Machine Learning Foundations

Introduction to supervised/unsupervised learning, and concepts like training and testing.

Weeks 7-8: Building Predictive Models

Implementing linear regression and logistic regression models with Scikit-learn and a final capstone project.
